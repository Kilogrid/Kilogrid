#define CANSPEED_125   3

BRP<5:0> bits of the CNF1 configuration register The Bitrate prescaler bits determine the nominal time quantum of the chip, and, therefore, the bit time. –> TQ = (1 + CANSPEED) / FOSC Note: FOSC = 8 MHz for the module and dispatcher. See the init_mcp2515 function for more detail on the configuration of the bitrate of the CAN controller.
